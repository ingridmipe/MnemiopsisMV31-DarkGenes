# Ctenophore-DarkGenes
This repo contains the code for the analyses of Mnemiopsis leidyi "dark genes".

## Project Structure

```
Ctenophore-DarkGenes/
├── bin/         # Code and scripts for analysis
├── data/        # Input data files
├── results/     # Analysis outputs and results
├── LICENSE      # MIT License
└── README.md    # This file
```

### Directory Descriptions

- **bin/**: Contains all bioinformatics analysis scripts, pipelines, and utility functions
- **data/**: Stores input data including raw sequencing data, reference genomes, and annotation files
- **results/**: Contains analysis outputs such as figures, tables, processed data, and reports

## Getting Started

1. Clone this repository
2. Place your analysis scripts in the `bin/` directory
3. Add your input data to the `data/` directory
4. Results will be generated in the `results/` directory

## Notes

- Large data files and results should not be committed to the repository
- See `.gitignore` for excluded file patterns
